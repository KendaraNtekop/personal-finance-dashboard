# Personal Finance Dashboard

A responsive personal finance web app built with vanilla HTML, CSS and JavaScript.
Live demo: [Click here]([YOUR GITHUB PAGES LINK])

## What it does

- Track income and expenses with category tagging
- View real-time balance, total income and total spent
- See spending breakdown by category with visual progress bars
- Set savings goals and track progress toward each target
- Single-page navigation across four views — Overview, Transactions, Savings and Expenses

## Built with

- HTML5
- CSS3 (CSS Grid, Flexbox, CSS Variables)
- Vanilla JavaScript (DOM manipulation, array methods, dynamic rendering)

## What I learned

Building this project taught me how to structure a JavaScript application
from scratch — managing data with arrays and objects, dynamically rendering
HTML from data, handling user interactions with event listeners, and
organising a single-page app with a show/hide view system.
